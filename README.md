# Value Added Course - MERN Full Stack Development

This repository contains all the hands-on practice programs, assignments, mini projects, and React applications developed during the **Value Added Course on MERN Full Stack Development** conducted at our college.

## 📚 Course Overview

The purpose of this repository is to maintain all practical work completed throughout the course and to track learning progress.

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- React.js
- Vite
- Git & GitHub


## 🚀 Projects Included

- SBJITMR Admission Portal (React + Vite)
- Reusable React Components
- Student Registration Form
- Form Validation using React
- React Hooks (`useState`)
- Responsive User Interface

## 🎯 Learning Outcomes

During this course, we are learning:

- React Fundamentals
- Functional Components
- Props
- State Management (`useState`)
- Event Handling
- Form Validation
- Component Reusability
- Responsive Web Design
- Git & GitHub Version Control
